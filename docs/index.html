Scandit Data Capture SDK Documentation for Cordova#

Welcome to the Scandit Data Capture SDK Developer Documentation for Cordova. If you are trying out the SDK or you are already integrating it into an app, you’ve come to the right place.

Warning

The Scandit Data Capture SDK for Cordova is currently in beta. The stable release is planned for late 2019.

Known issues:

  • DataCaptureContext instances are not safe to be created shortly after an already existing context was used

  • DataCaptureContext updates might be delayed by BarcodeCaptureListener callbacks

  • DataCaptureView instances might not hide or resize as expected under certain circumstances
